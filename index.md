### First install

Need third party apps : 

+ NPM
+ Git
+ Mongo
+ Mongod
+ Express JS
+ Nodemon

Installation : 

+ start mongod

Open npm/git on project folder right click select git bash here
and run : 

+ npm i
+ nodemon

Open localhost:5000
